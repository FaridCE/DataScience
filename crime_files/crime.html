ACP para encontrar patrones delictivos

Usando datos del INEGI

Datos a analizar: Homicidio doloso, Robo de vehculo automotor, Lesiones dolosas, Extorsin; por Entidad.

Limpieza de datos

El archivo csv se encontra dividido por columnas en donde se repite el ao, la entidad, y el tipo de delito.

  1. Primero se agrupa por Subtipo de delito (eliminando las columnas innecesarias) sumando el nmero de crmenes por mes
  2. Luego hacemos una mscara booleana para obtener los subdelitos de inters (Homicidio doloso, etc).
  3. Por ltimo, promediamos los delitos anuales y pivoteamos (agrupamos en un multi-ndice) el marco de datos para que aparezcan los subdelitos anualizados por Ao y Entidad.
In[107]:
import pandas as pd
from sklearn.preprocessing import StandardScaler
import numpy as np

df = pd.read_csv('Data/delitos.csv', encoding='latin1')
df.drop(['Clave_Ent', 'Bien jurdico afectado', 'Tipo de delito', 'Modalidad'], axis=1, inplace=True)
# Columns that contain the numeric month values
month_cols = ['Enero', 'Febrero', 'Marzo', 'Abril', 'Mayo', 'Junio',
              'Julio', 'Agosto', 'Septiembre', 'Octubre', 'Noviembre', 'Diciembre']

# 1 Group by the subtype of crime and sum the monthly values
df_sub = (
    df.groupby(['Ao','Entidad','Subtipo de delito'])[month_cols]
      .sum()
      .reset_index()
)
# 2 Values you want to keep
SubDelitos = [
    'Homicidio doloso',
    'Robo de vehculo automotor',
    'Lesiones dolosas',
    'Extorsin'
]

# Boolean mask and filter
mask   = df_sub['Subtipo de delito'].isin(SubDelitos)
filtered_df = df_sub[mask].reset_index(drop=True)

# 3. Sumar la incidencia mensual para obtener el total anual

df_anual = filtered_df.groupby(['Ao', 'Entidad', 'Subtipo de delito'])[month_cols].sum().reset_index()

# Pivotear para que los subtipos de delito sean columnas
df_pivot = df_anual.pivot_table(
    index=['Ao', 'Entidad'],
    columns='Subtipo de delito',
    values= month_cols[0] # Usar cualquier mes para el pivot, ya que todos tienen la suma
).reset_index()

# El resultado 'df_pivot' tendr columnas: ['Ao', 'Entidad', 'Extorsin', 'Homicidio doloso', 'Lesiones dolosas', 'Robo de vehculo automotor']
# df_pivot # Descomentar para ver el DataFrame resultante

# Seleccionar solo las columnas de los delitos para la matriz de clculo
X = df_pivot[['Extorsin', 'Homicidio doloso', 'Lesiones dolosas', 'Robo de vehculo automotor']]

Pre-anlisis de datos

Para analizar los datos por componentes principales hay que:

  1. Estandarizar los datos: escribirlos de manera relativa a su valor promedio y centralizarlos. Solo nos interesan las fluctuaciones.
  2. Correlacionar los datos: escribir la matriz de correlacin para cada Entidad.

Anlisis por Componentes Principales

Clculo y Extraccin de Componentes Principales

Usaremos el mdulo PCA de scikit-learn para obtener los componentes, la varianza explicada y las cargas factoriales.

A. Aplicacin del Modelo PCA

Continuando con el DataFrame estandarizado (X_scaled) de los cuatro delitos:

B. Anlisis de la Varianza Explicada (Valores Propios)

El primer paso de la interpretacin es determinar cunta informacin (varianza) de los datos originales es capturada por cada componente.

In[116]:
# Varianza Explicada (Eigenvalues o Valores Propios)
varianza_explicada = pca.explained_variance_

# Proporcin de la Varianza Explicada
proporcion_varianza = pca.explained_variance_ratio_

# Varianza Acumulada
varianza_acumulada = np.cumsum(proporcion_varianza)

# Crear un DataFrame para la visualizacin
df_varianza = pd.DataFrame({
    'Componente': range(1, len(varianza_explicada) + 1),
    'Valor Propio': varianza_explicada,
    'Proporcin Explicada': proporcion_varianza,
    'Varianza Acumulada': varianza_acumulada
})

Determinacin del Nmero de Componentes (Patrones)

Usando los criterios comunes:

  1. Criterio de Kaiser (Valor Propio > 1): Se mantienen los componentes cuyo valor propio es mayor a 1. En este ejemplo, sera solamente PC1.
  2. Varianza Acumulada: Se mantienen los componentes necesarios para alcanzar un umbral razonable (ej. 80-90%). En este ejemplo, con PC1 y PC2 se explica el 91.3% de la varianza total, lo cual es muy bueno.

El anlisis se centrara, por lo tanto, en los dos primeros componentes (PC1 y PC2).

Interpretacin de los Componentes (Cargas Factoriales)

Los vectores propios (o components_ en scikit-learn) son la clave para entender el patrn. Se les conoce como Cargas Factoriales (Loadings) y representan la correlacin entre cada delito original y cada componente principal.

# Los vectores propios se encuentran en el atributo 'components_'

Resumen de los Patrones (Componentes)

Al reducir la dimensionalidad de la matriz, el ACP ha encontrado los siguientes cuatro patrones de riesgo distintos en los datos del INEGI:

  1. PC1: Riesgo General (Mide el volumen total de la delincuencia en la entidad).
  2. PC2: Riesgo de Homicidio (Mide el peso del Homicidio dentro de la estructura criminal).
  3. PC3: Riesgo de Robo vs. Extorsin (Contrasta el patrn de robo de vehculos con el de extorsin).
  4. PC4: Riesgo de Lesiones (Mide el peso de la violencia interpersonal no letal).

Visualizacin de los Resultados (Biplot)

El Biplot es la herramienta de visualizacin ms til en el ACP, ya que permite ver tres cosas simultneamente:

  1. Las Observaciones: La posicin de cada Entidad-Ao (los puntos en el grfico).
  2. Las Variables: La direccin y magnitud de las variables originales (los delitos) en el nuevo espacio de componentes (los vectores).
  3. La Relacin: Cmo se relacionan las variables entre s y con las observaciones.

Interpretacin de la Visualizacin

  • Vectores Cercanos: Delitos cuyos vectores apuntan en la misma direccin (ej. Extorsin, Lesiones y Robo) tienen una alta correlacin positiva.
  • Vectores Opuestos: Si un vector apunta en la direccin opuesta (separados 180), tienen una alta correlacin negativa (no ocurre en este caso, pero PC3 s mostr esto).
  • Puntos (Entidades):
    • Los puntos a la derecha del grfico tienen puntuaciones altas en PC1 (alto riesgo delictivo general).
    • Los puntos en la parte superior del grfico tienen puntuaciones altas en PC2 (alta incidencia desproporcionada de Homicidio Doloso).
  • Distancia: La longitud del vector indica la varianza del delito que explica ese componente (mientras ms largo, ms importante para ese plano).
